# Frontend Test Assignment

## Overview
A responsive front-end implementation built from a provided Figma design.  
This project demonstrates strong fundamentals in HTML, CSS, and JavaScript with a focus on layout accuracy, clean structure, and responsive behavior across devices.

## Tech Stack
- HTML5
- CSS3
- JavaScript (Vanilla JS)

## Key Features
- implementation from Figma
- Fully responsive layout (desktop, tablet, mobile)
- Semantic HTML structure
- Clean and maintainable CSS
- Interactive UI elements using JavaScript
- Cross-browser compatibility

## What I Focused On
- Accurate spacing, typography, and alignment
- Reusable CSS structure
- Responsive breakpoints
- Clean file organization
- Smooth hover and transition effects
